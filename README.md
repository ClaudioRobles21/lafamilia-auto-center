# 🚗 La Familia Auto Center - Página Web

Página web interactiva para La Familia Auto Center con:

- ✅ **Galería de autos** — 5 vehículos con especificaciones completas
- ✅ **Chatbot inteligente** — Respuestas automáticas sobre los autos
- ✅ **Agendamiento de citas** — Integrado con Google Form
- ✅ **Diseño responsive** — Se adapta a móvil y escritorio
- ✅ **Contacto directo** — Teléfono y Facebook

## 📋 Cómo usar

### Opción 1: Clonar y servir localmente
```bash
git clone https://github.com/TU_USUARIO/lafamilia-auto-center.git
cd lafamilia-auto-center
# Abre index.html en tu navegador
```

### Opción 2: GitHub Pages (acceso desde cualquier dispositivo)
1. Sube este repositorio a GitHub
2. Ve a Settings → Pages
3. Selecciona `main` branch como fuente
4. Tu página estará disponible en: `https://TU_USUARIO.github.io/lafamilia-auto-center`

## 🔧 Características

- **Chatbot sin tokens** — Respuestas automáticas, cero costo
- **Google Form integrado** — Recibe solicitudes automáticamente
